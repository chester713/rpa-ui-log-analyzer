# Pattern: Set Value

## Action
set

## Object
value

## Method
DOM Manipulation (Web) / UIA Tree Manipulation (Desktop) / Hardware Simulation (Visual)

## Category
Modification

## Contexts
- web
- desktop
- visual

## Description
Set a value to an input field. This includes typing text, pasting content, or clearing and entering new text. In web environments, DOM manipulation is used. In desktop environments, UIA tree manipulation is used. In visual environments, hardware simulation is used.
