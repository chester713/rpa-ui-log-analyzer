# Pattern: Select Option

## Action
select

## Object
option

## Method
DOM Manipulation (Web) / UIA Tree Manipulation (Desktop) / Hardware Simulation (Visual)

## Category
Modification

## Contexts
- web
- desktop
- visual

## Description
Select an option from a dropdown, list, or combo box. This includes clicking to open the dropdown and selecting a specific item. In web environments, DOM manipulation is used. In desktop environments, UIA tree manipulation is used. In visual environments, hardware simulation is used.
