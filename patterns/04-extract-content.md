# Pattern: Extract Content

## Action
extract

## Object
content

## Method
DOM Parsing (Web) / UIA Tree Parsing (Desktop) / Visual Recognition (Visual)

## Category
Extraction

## Contexts
- web
- desktop
- visual

## Description
Extract text or content from a UI element. This includes getting text from text fields, labels, buttons, or other display elements. In web environments, DOM parsing is used. In desktop environments, UIA tree parsing is used. In visual environments, visual recognition is used.
