# Pattern: Activate Element

## Action
activate

## Object
element

## Method
DOM Manipulation (Web) / UIA Tree Manipulation (Desktop) / Hardware Simulation (Visual)

## Category
Modification

## Contexts
- web
- desktop
- visual

## Description
Activate a UI element such as a button, link, or menu item. This pattern triggers the primary action associated with the element. In web environments, DOM manipulation is used. In desktop environments, UIA tree manipulation is used. In visual environments, hardware simulation is used.
